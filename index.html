Conta Google & Backup
Login real com Google

Para logar de verdade (sem simulação), o Google exige uma credencial gratuita chamada Client ID — é assim que funciona o login do Google em qualquer site, não tem como pular essa parte por segurança. Leva ~2 minutos e não precisa de servidor:
  1. Acesse console.cloud.google.com/apis/credentials
  2. Crie um "ID do cliente OAuth" → tipo "Aplicativo da Web"
  3. Em "Origens JavaScript autorizadas", adicione o endereço onde este site fica salvo/hospedado (ex: o link do GitHub Pages)
  4. Copie o Client ID gerado e cole no campo abaixo — ou cole-o direto na constante GOOGLE_CLIENT_ID no topo do script, e essa tela de configuração não aparece mais para ninguém: o botão "Entrar com Google" já aparece pronto, igual nos outros sites
